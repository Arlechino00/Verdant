# Verdant 🌿

Verdant is a modern Android application designed for plant enthusiasts and nature explorers. It serves as a comprehensive plant encyclopedia and an intelligent identification tool, helping users discover and learn about the flora around them.

## Features

-   **🔍 AI-Powered Identification:** Leverage machine learning to identify plants instantly using your device's camera. Powered by Google ML Kit.
-   **📚 Extensive Plant Library:** Explore a rich database of plants, including detailed descriptions, medicinal properties, and toxicity warnings.
-   **🔐 Secure Authentication:** Create an account to personalize your experience. Powered by Firebase Authentication.
-   **🔎 Advanced Search:** Quickly find information about specific plants using the built-in search functionality.
-   **🎨 Modern Design:** A beautiful, responsive user interface built with Jetpack Compose and Material 3, featuring custom typography with the Poppins font family.

## Tech Stack

-   **Language:** [Kotlin](https://kotlinlang.org/)
-   **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
-   **Architecture:** MVVM (Model-View-ViewModel)
-   **Dependency Injection:** [Hilt](https://developer.android.com/training/dependency-injection/hilt-android)
-   **Backend & Auth:** [Firebase](https://firebase.google.com/)
-   **Machine Learning:** [Google ML Kit (Vision)](https://developers.google.com/ml-kit/vision/image-labeling)
-   **Image Loading:** [Coil](https://coil-kt.github.io/coil/)
-   **Navigation:** [Compose Navigation](https://developer.android.com/jetpack/compose/navigation)

## Getting Started

### Prerequisites

-   Android Studio Jellyfish or newer.
-   JDK 17 or higher.
-   A Firebase project (you'll need to provide your own `google-services.json`).

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/Delia/Verdant.git
    ```
2.  Open the project in Android Studio.
3.  Add your `google-services.json` file to the `app/` directory.
4.  Sync the project with Gradle files.
5.  Build and run the app on an emulator or physical device.
